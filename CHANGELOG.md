# Changelog

All material specification and archive-structure changes are recorded here. Raw capture history remains in Git and is not rewritten to make later designs appear original.

## v1.1 — 2026-07-18

Authored as an addendum by Claude Opus 4.8 with Jaron Kyler Bragg; consolidated into the live specification by Codex.

- Replaced enforced synthesis isolation with declared context.
- Split convergence into independent and informed agreement/disagreement modes.
- Added one byte-identical prompt for both synthesizers.
- Added longitudinal signal tracking, lifecycle states, cycle health, and stagnation measurement.
- Split unavailable attempts from content refusals and corrected the known Kimi AI / Tech attempt sequence.
- Expanded the reading from seven sections to nine.
- Forward-declared the planned financial-layer migration.
- Preserved the addendum’s credential language verbatim while clarifying in the live spec that a read-only key can still leak confidential account data and incur rotation cost even without trading or withdrawal authority.
- Added two implementation extensions needed for durable public use: stable harness-assigned signal IDs and an append-only JSONL event ledger from which the human tracker can be reproduced.
- Added JSON Schemas and synthesis JSON sidecars so future automation and the public website share one machine contract instead of parsing prose as state.

The supplied addendum is preserved verbatim in [SPEC-ADDENDUM-v1.1.md](SPEC-ADDENDUM-v1.1.md).
