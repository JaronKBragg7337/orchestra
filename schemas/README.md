# Machine-readable contracts

These JSON Schemas keep the synthesizers, comparison harness, tracker, and future public website on one contract:

- [synthesis-item.schema.json](synthesis-item.schema.json)
- [synthesis-output.schema.json](synthesis-output.schema.json)
- [comparison.schema.json](comparison.schema.json)
- [tracking-event.schema.json](tracking-event.schema.json)

Markdown remains the human record. Structured outputs validated against these schemas are the automation boundary. Schema versions are explicit and changes require a specification version or documented backward-compatible amendment.
